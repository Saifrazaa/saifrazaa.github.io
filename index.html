As an Software Engineer, I develop hybrid mobile applications for Android and iOS platforms using React Native, as well as web applications using React.js, styled-components, and JavaScript. I have successfully deployed multiple mobile apps on Playstore and Apple Store, reaching thousands of users and generating positive feedback.

I also design and implement user-friendly interfaces and web pages using HTML, CSS, and JavaScript, ensuring that they are responsive and work across multiple devices and screen sizes. I collaborate with back-end developers to integrate front-end components with back-end functionality, using REST APIs and Express.js. I write clean, maintainable, and efficient code, and communicate and work effectively with designers, project managers, and other stakeholders.

I have 4+ years of experience in mobile and web application development, having previously worked as a Fullstack Developer Web and Mobile Apps at Panacloud, where I used React.js, Redux, Node.js, MongoDB, Firebase, and React Native to create dynamic and interactive web and mobile apps. I also completed an internship as a Web Application Developer at Panacloud, where I learned the basics of React.js and Redux.

I hold a Bachelor's degree in Computer Software Engineering from Mohammad Ali Jinnah University Karachi, Pakistan, and I have obtained certifications in MongooseJS Essentials, Git Started with GitHub, and Javascript Public Test. I am passionate about staying up-to-date with the latest industry trends and technologies, and I am always eager to learn new skills and tools to enhance my performance and deliver high-quality solutions.
